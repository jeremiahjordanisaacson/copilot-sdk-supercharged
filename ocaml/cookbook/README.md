# GitHub Copilot SDK Cookbook - OCaml

Short, practical recipes for using the GitHub Copilot SDK with OCaml. Each recipe is a runnable `.ml` module, copy-pasteable, and production-ready.

## Recipes

- [Error Handling](error_handling.ml): Handle errors gracefully including connection failures, timeouts, and cleanup.
- [Custom Tools](custom_tools.ml): Define custom tools and register them with the SDK.
- [Model Selection](model_selection.ml): List available models and select specific ones for sessions.
- [Multi-Turn Chat](multi_turn.ml): Build multi-turn conversations with context retention.
- [Permission Handling](permission_handling.ml): Handle permission requests from the assistant.
- [Streaming](streaming.ml): Subscribe to and process streaming events in real time.

## Contributing

Add a new recipe by creating a `.ml` file in this folder and linking it above.
