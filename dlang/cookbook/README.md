# GitHub Copilot SDK Cookbook - D

Short, practical recipes for using the GitHub Copilot SDK with D. Each recipe is a runnable `.d` module, copy-pasteable, and production-ready.

## Recipes

- [Error Handling](error_handling.d): Handle errors gracefully including connection failures, timeouts, and cleanup.
- [Multiple Sessions](multiple_sessions.d): Manage multiple independent conversations simultaneously.
- [Persisting Sessions](persisting_sessions.d): Save and resume sessions across restarts.
- [Tool Usage](tool_usage.d): Define custom tools and register them with the SDK.
- [Streaming](streaming.d): Subscribe to and process streaming events in real time.
- [Advanced Config](advanced_config.d): Per-session auth, SessionFs, commands, system prompts, and more.

## Contributing

Add a new recipe by creating a `.d` file in this folder and linking it above.
