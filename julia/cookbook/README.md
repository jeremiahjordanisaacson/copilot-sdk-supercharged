# GitHub Copilot SDK Cookbook - Julia

Short, practical recipes for using the GitHub Copilot SDK with Julia. Each recipe is a runnable `.jl` script, copy-pasteable, and production-ready.

## Recipes

- [Error Handling](error_handling.jl): Handle errors gracefully including connection failures, timeouts, and cleanup.
- [Concurrent Sessions](concurrent_sessions.jl): Manage multiple independent conversations simultaneously using tasks.
- [Custom Tools](custom_tools.jl): Define custom tools and register them with the SDK.
- [External Server](external_server.jl): Connect to an external Copilot CLI server instead of spawning a local process.
- [Multi-Turn Chat](multi_turn.jl): Build multi-turn conversations with context retention.
- [Permission Handling](permission_handling.jl): Handle permission requests from the assistant.
- [Streaming Chat](streaming_chat.jl): Subscribe to and process streaming events in real time.

## Contributing

Add a new recipe by creating a `.jl` file in this folder and linking it above.
